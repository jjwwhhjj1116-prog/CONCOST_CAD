2단계 · 도면 정리 / QTO 레이어
원본 블록 구조는 유지한 채 삭제 후보 레이어만 정리하고, 적산용 QTO 표준 레이어 골격을 만듭니다.
⚠ 먼저 1단계 분석을 완료하세요.
🧹

위 도면 정리 실행 버튼을 누르면 결과가 여기에 표시됩니다.

3단계 · 실 추출 (실명 · 면적 · 둘레)
내부 마감 작업 도면(실 경계 폴리라인)에서 실코드·실명·면적(A)·둘레(L)를 자동 추출합니다. FIN 내부 입력 변수와 직결됩니다.
🏗️
내부 마감 작업 도면(DWG/DXF)을 드래그하거나 클릭하여 업로드
사람이 실 경계 폴리라인을 그린 작업 도면 (예: 100-A- 평면도)
📐

작업 도면을 올리고 실 추출 실행을 누르면 실별 면적·둘레 표가 나옵니다.

4단계 · 실 면적 산출
실 경계와 실명 매칭, 실별 면적 산출.
